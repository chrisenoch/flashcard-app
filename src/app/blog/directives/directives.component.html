Tour Guide Demo

Overview

  • It is a tooltip with extra features such as timers and the ability to move from one element to another.

Features

  • Can move from one element to another.
  • Automatically adjusts when:
    • Screen is resized.
    • User scrolls.
    • Layout shifts due to an item appearing in the DOM that moves content / brreaks the layout.
  • Customise the arrows.
  • Include any content you want via ng-content.

Limitations

  • Does not adjust position if it overflows the container/page.
